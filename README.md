# React-Native-Demo 
项目克隆下来，进入项目目录，执行
```
yarn
react-native run-android  // 运行安卓，前提是先启动模拟器
react-native run-ios  // 运行iOS，直接运行就行，前提是使用安装了Xcode的Mac
```
